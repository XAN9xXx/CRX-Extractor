How to use CRX Extractor?

  1. Drop the .crx file into the placeholder
  2. Wait until it gets processed
  3. Click "Download" and get source code of Chrome extension as .zip archive with all related metadata

How to get .crx Chrome Extension File?

  1. Head your browser to Chrome WebStore
  2. Find an extension you'd like to save and copy the URL from the address bar of your browser
  3. Paste the link to the right pane on CRXExtractor and click "OK". Then click the appeared button. The download should start immediately.
