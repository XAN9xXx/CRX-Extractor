# CRX-Extractor

A web tool to download .crx files, extract source code, and review privacy/security risks.

Forked from [vladignatyev/crx-extractor](https://github.com/vladignatyev/crx-extractor).
Removed promotional content to keep clean.

## Usage

**Prerequisites:** Node.js ≥ 18

```bash
npm install
npm run dev      # starts at http://localhost:3000
```
